#ec2-shrink-ebs-root-volume

This Script Shrinks/Reduce ec2 ebs root volume size.

Requirements
------------
ec2= Access Key 
ec2= Secret Key

For managing ec2 API Access.

Dependencies
------------
None

Example
-------
./ec2-shrink-ebs ip size

ip = ec2 instance private ip.
size = size in GB.
